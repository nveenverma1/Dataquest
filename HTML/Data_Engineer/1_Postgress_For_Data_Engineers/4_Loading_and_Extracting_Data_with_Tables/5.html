Using copy_expert(), we can also extract a Postgres table to any Python file object. To write the file out, we need to use the COPY ... TO command and write it out to STDOUT. Then, we pass in the file object as the second argument to save the file.

xxxxxxxxxx
 
cur = conn.cursor()
with open('example.txt', 'w') as f:
    cur.copy_expert('COPY ign_reviews TO STDOUT', f)
instructions
  • Use the provided cur variable.
  • Open a old_ign_reviews.csv file using the statement with open() as f.
  • Execute the COPY ... TO method on the old_ign_reviews table using the copy_expert method.
    • Add the CSV and HEADER options.
    • Write it out to the old_ign_reviews.csv file.
